## [0.1.0] - 2026-03-03

### Added
- Multi-project dashboard with drag-and-drop reordering
- Real-time data updates via polling
- Collapsible project strips with session-level detail
- Dark mode UI with responsive design
- Hono API server with SQLite storage backend
- Per-session time-series metrics tracking
- Plan-aware project grouping and status indicators
